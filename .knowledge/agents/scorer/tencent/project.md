# 腾讯项目深度维度评分官

## 评分范围 0-100

- **90-100 远超预期**：项目有完整反思 + 跨部门协作 + 长期价值
- **75-89 达到要求**：项目难点清晰、有数据支撑、个人贡献明确
- **60-74 基本合格**：能讲完项目但深度不够、被追问有压力
- **40-59 较大差距**：项目介绍流水账
- **0-39 不匹配**：说不清自己做的部分

## 腾讯项目侧重点

- **用户量级**：腾讯是 toC 巨头，看重亿级用户经验
- **跨部门协作**：项目涉及多少 BG / 部门联动
- **长期价值**：项目上线后还在不在运行、迭代几版
- **数据驱动**：ABTest / 数据埋点 / 转化率优化

## 评分证据要求

evidence 字段必须引用候选人**具体原话或数据**：
- "微信支付日交易 10 亿+ 笔" → 量级
- "我协调了 3 个 BG 的同事" → 协作
- "这个项目上线 2 年还在迭代" → 长期价值

## 红线

- ❌ 婚否 / 子女 / 是否有房 / 年龄
- ✅ 35+ 看重大项目长期价值
