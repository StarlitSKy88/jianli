## 1. 产品定位（一句话）

**面试陪练** = 用户上传简历 → 选公司/岗位 → AI 模拟真实面试官提问 → 多 Agent 联合评分 → 出局报告

---

## 2. 用户场景

### 2.1 MVP 阶段目标用户
- **核心**：35+ 已失业或想转型的程序员
- **使用动机**：练习软技能 + 应对大厂真实面试场景
- **使用频率**：建议每周 2-3 次（每天最多 3 次免费，第 4 次起 ¥9.9/次）

### 2.2 核心使用流程
```
1. 注册（邮箱 + 密码）
2. 上传简历（PDF / 在线填表）
3. 选择面试公司 + 岗位 + 难度（4×N）
4. 进入"AI 面试间"（与画像化的 AI 面试官对话 20 分钟）
5. 面试中：AI 提问 → 用户语音/文字回答 → AI 追问
6. 面试结束 → 多 Agent 联合评分 → 出局报告
7. 报告包含：
   - 5 维分数（每公司独立定义）
   - 与同岗位 TOP 10% 的差距分析
   - 提问回放（文字）
   - 改进建议
```

## 3. MVP 功能范围（YAGNI）

### 3.1 ✅ MVP 必须做
- 注册登录（邮箱+密码）
- 简历上传（PDF 解析）
- 4 家公司画像（字节/阿里/腾讯/B站） × 4 类岗位 = **16 关**
- AI 模拟面试（文字）
- 多 Agent 联合评分（3 个评分 Agent + 1 个汇总 Agent）
- 局后报告
- 次数限制 + 付费（¥9.9/次）
- 5 个最关键埋点

### 3.2 ❌ MVP 不做
- 视频面试（仅录音 / 文字）
- 真人面试官匹配
- 完整职业规划
- 企业 BD 服务（To-B）
- 多语言（仅中文）
- 移动端 App（仅 Web）
- **录音上传 / 真人复盘**（用户决策删除 — 大幅简化 MVP）

## 4. AI 面试官画像（差异化 — Task 调研基础）

### 4.1 字节跳动面试官
- **人设**：技术深度派、节奏快、不废话、问必刨根问底
- **5 维评分**（独立权重）：
  - 算法与数据结构 30%
  - 计算机基础 20%
  - 项目与工程 25%
  - 系统设计 15%
  - 文化匹配（聪明/皮实/自省/乐观）10%
- **复盘 Prompt**：
  > "你是字节跳动 3-2 资深面试官，请严格按上方权重评估，语气犀利、客观、避免虚词。"

### 4.2 阿里巴巴面试官
- **人设**：价值观深挖派、善于讲故事、决策逻辑派
- **5 维评分**：
  - 业绩贡献 30%
  - 六脉神剑 30%
  - 技术深度 20%
  - 业务理解 10%
  - 协作与影响力 10%
- **复盘 Prompt**：
  > "你是阿里 P8 + HR 政委联合面试官，重点判断价值观匹配度，用 STAR 法则追问决策逻辑，语气温和但刨根问底。"

### 4.3 腾讯面试官
- **人设**：抗压测试派、实战复盘派
- **5 维评分**：
  - 技术能力 30%
  - 项目复盘 25%
  - 抗压与韧性 15%
  - 沟通与协作 15%
  - 价值观与稳定性 15%
- **复盘 Prompt**：
  > "你是腾讯 T3 面试官 + HRBP，重点考察压力下的决策逻辑与事后复盘深度，语气沉稳。"

### 4.4 B 站面试官
- **人设**：场景化、氛围轻松、拒绝八股
- **5 维评分**：
  - 内容场景理解 25%
  - 技术能力 25%
  - 社区文化契合 20%
  - 沟通与表达能力 15%
  - 创新与主动性 15%
- **复盘 Prompt**：
  > "你是 B 站主站运营负责人，重点考察候选人是否'理解社区'与'有创意'，语气轻松但有锐度。"

## 5. 核心体验细节

### 5.1 输入流
- **简历**：PDF 上传 + 字段化抽取
- **公司**：4 家单选
- **岗位**：
  - 字节：后端 / 前端 / 算法 / 产品
  - 阿里：后端 / 数据 / 产品 / 运营
  - 腾讯：后端 / 客户端 / 产品 / 运营
  - B站：后端 / 运营 / 内容审核 / UP 主运营
- **难度**：初级 / 中级 / 高级

### 5.2 输出流（评分报告）
```
【总分】85 分（位于同级别 TOP 15%）

【分维度】
- 算法：82（中等偏上）
- 计基：90（优秀）
- 项目：80（中等，缺少数据化表达）
- 系统设计：85（良好）
- 文化：91（优秀）

【改进建议 TOP 3】
1. 项目复盘：未来回答请按 STAR 法则 + 数据化
2. 算法：欠缺 Hard 题，需补 DP + 图论
3. 加分项：可主动展示你的"自省"故事
```

## 10. 隐私与合规

### 10.1 严守法律红线
- ❌ 不询问婚否 / 有子女 / 是否有房（《就业促进法》第 27 条）
- ❌ 不询问任何与岗位能力无关的个人信息
- ❌ 不存储身份证号、银行卡号、详细住址

### 10.2 数据安全
- 密码 bcrypt 加密
- AI 调用零保留策略
- 数据库 TDE 加密（生产环境）

## 12. MVP 验收标准

| 项 | 验收点 |
|---|---|
| 注册登录 | 邮箱 + 密码 + 验证邮件 |
| 简历上传 | PDF 解析成功率 > 90% |
| 16 关面试 | 4 公司 × 4 岗位 |
| 面试对话 | 单场 15-20 分钟，至少 5 轮问答 |
| AI 评分 | 3 Agent 评分一致性 > 0.7（Kappa 系数） |
| 评分报告 | 总分 + 分维度 + 改进建议 + 同级别差距 |
| 付费 | 微信支付能跑通 |
| 埋点 | 5 个事件全部埋好 |
